# Python_Tkinter_SQLite_CashLogger
**CashLogger: Personal Expense Tracker**

A desktop GUI application designed to help users track their personal finances effectively. Built using Python and Tkinter, this project demonstrates the implementation of a relational database system to manage user income and expenses.

Project Overview

Language: Python

GUI Framework: Tkinter

Database: SQLite3

Visualization: Matplotlib

Type: Desktop Application
